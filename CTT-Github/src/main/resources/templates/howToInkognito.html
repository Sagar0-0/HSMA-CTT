Inkognito im Firefox

Wussten Sie schon, man kann mehrere Personen von einem Gerät aus anmelden! Dazu müssen Sie lediglich einen neuen Inkognito-Tab auf ihrem Endgerät öffnen.

Drücken Sie zuerst auf das Tab Symbol im Browser, dann auf das Maskensymbol (Privates Surfen)
